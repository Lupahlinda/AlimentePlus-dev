# Alimente+

Projeto acadêmico da Turma de ADS do IESB – Campus Ceilândia para a disciplina Engenharia de Software e Métodos Ágeis (Profª Adriana Falcomer Pontes).

## Como rodar
- **Pré-requisitos**: Node 18+, npm 9+
- **Instalar**: `npm install`
- **Mock API**: `npm run mock:server` (http://localhost:3001)
- **Dev**: `npm run dev` (http://localhost:5173)

## Scripts úteis
- **Lint**: `npm run lint` | **Fix**: `npm run lint:fix`
- **Format**: `npm run format`
- **Build/Preview**: `npm run build` / `npm run preview`

## Tecnologias
React, Vite, Tailwind, React Router, Axios, Leaflet/React-Leaflet, ESLint/Prettier, json-server. Veja `package.json`.

## Estrutura básica
- `src/` código-fonte
- `public/` estáticos
- `db.json` dados mock
